# Bekritsky_masterthesis_batches_and_scripts
In this repository you can find all the R scripts and SPM 12 batches (converted to .m files) I used for the statistical analyses in my master thesis.
All scripts were conducted in SPM12 or R Studio, version 4.2.1, and optimized with the help of Claude AI and ChatGPT.
Please note that you need to enter paths and subject IDs on your own.
